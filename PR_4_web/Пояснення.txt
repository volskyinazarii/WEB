Ця програма використовує геолокаційний API браузера для отримання поточного місцезнаходження користувача.

Створює основну HTML-структуру з двома блоками: "location" та "distance".
Визиває скрипт error.js.

Додає подію "DOMContentLoaded", яка викликає функцію getMyLocation при завантаженні сторінки.
Функція getMyLocation перевіряє, чи браузер підтримує геолокацію.
Якщо так, то вона викликає navigator.geolocation.getCurrentPosition з двома функціями зворотного виклику: displayLocation при успішному отриманні місцезнаходження та displayError при помилці.
Функція displayLocation:

Отримує координати користувача (широту та довготу) та точність вимірювання (accuracy).
Виводить отримані дані у блок з id "location".
Викликає функцію computeDistance для розрахунку відстані між поточним місцезнаходженням та фіксованим місцем (у цьому випадку, координатами Коледжу).
Виводить отриману відстань у блок з id "distance".
Функція displayError:

Отримує об'єкт помилки та виводить відповідне повідомлення у блок з id "location".
Функція computeDistance:

Розраховує відстань між двома точками на поверхні Землі за допомогою формули гаверсинуса.
Приймає координати початкової точки (startCoords) та фіксованої точки (destCoords).
Виводить результат у кількості кілометрів.
Отримані дані виводяться на веб-сторінку в блоках "location" та "distance". Якщо виникає помилка при отриманні геолокації, виводиться відповідне повідомлення у блок "location".
